# Front-utils
Um framework de comportamento e estilo.
Saiba todas as features que te ajudarão a produzir um front-end mais rápido.

### LAYOUT
Aprenda sobre o GRID e nossas Classes Funcionais.

### COMPONENTES
Saiba como manipular todos os componentes como: Abas, Alertas, Barra de Progresso, Botões, Collapse &amp; Accordeon, Menu Responsive, Modal, Tabelas, Tooltip e muito mais.

### Formulários
Saiba como manipular os campos e grupos de formulários com o Front-utils.

#### Instale usando Bower [bower](http://bower.io)

Para instalar o Front-utils utilizando o Bower é bem simples:

```sh
$ bower install https://github.com/leandro-mancini/front-utils.git
```
